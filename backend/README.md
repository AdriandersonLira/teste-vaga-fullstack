# Como rodar o projeto (parte do backend)?

#### Certifique der ter instalado

- npm@10.5.0
- node@20.11.1
- docker@
- postgres@16

##### 1 - Primeiro abra um terminal dentro da pasta backend e escreva

```bash
backend/$ npm i
```

##### 2 - Suba o container docker

```bash
backend/$ docker-compose up -d
```
